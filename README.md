# bank_note_authentification_model

This project is a machine learning model designed to classify banknotes as authentic or counterfeit. It uses statistical features extracted from images of banknotes and applies supervised learning techniques to perform binary classification.

The goal is to build a reliable system that can assist in detecting fake currency using data-driven methods.

PROBLEM STATEMENT

Counterfeit currency is a major issue in financial systems. Manual verification is prone to errors and inefficiency. This project applies machine learning to automate the authentication process by analyzing patterns in banknote data.

 DATASET

The dataset contains extracted features from images of banknotes:

 Variance of Wavelet Transformed image
 Skewness of Wavelet Transformed image
 Kurtosis of Wavelet Transformed image
 Entropy of image

TARGET VARIABLE:

 0 → Fake banknote
 1 → Genuine banknote

 TECHNOLOGIES USED:

 Python 
 Pandas
 NumPy
 Scikit-learn
 Matplotlib 
 
 MACHINE LEARNING MODEL

The model uses supervised learning classification algorithms such as:

 Logistic Regression
